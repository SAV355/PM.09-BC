# Банковский калькулятор

Сервис для расчёта ипотеки, автокредита, потребительского кредита и пенсионных накоплений.

## Технологии

- **Frontend**: React, Material-UI
- **Backend**: Node.js, Express, MongoDB
- **Почта**: Nodemailer (Gmail SMTP)
- **PDF**: jsPDF + html2canvas

## Установка и запуск

### Требования
- Node.js 16+
- MongoDB

### Установка зависимостей
```bash
# Клиент
cd client && npm install

# Сервер
