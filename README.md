# Банковский калькулятор

Сервис для расчета финансовых операций банка (ипотека, автокредит, пенсионные накопления).

## Функциональность

### Реализовано в MVP:
- ✅ Ипотечный калькулятор с полным расчетом по формулам из ТЗ
- ✅ Калькулятор автокредита с полным расчетом по формулам из ТЗ
- ✅ Потребительский кредит с полным расчетом по формулам из ТЗ
- ✅ Пенсионные накопления с полным расчетом по формулам из ТЗ
- ✅ Адаптивный дизайн с корпоративной цветовой схемой
- ✅ Сохранение расчетов в MongoDB
- ✅ Базовый административный интерфейс
- ✅ Страницы для других калькуляторов (заглушки)

### В разработке:

- ⏳ Отправка результатов на email
- ⏳ Полноценная админ-панель

## Технологии

### Фронтенд:
- React 18
- React Router 6
- Material-UI 5
- Axios для API запросов

### Бэкенд:
- Node.js + Express
- MongoDB + Mongoose
- Nodemailer (для email)

### Приведение всех калькуляторв к единому оформлению:

#### Теперь все 4 калькулятора оформленны в едином стиле:
🏠 `Ипотека` - синий (#1E3A8A) цвет <br>
🚗 `Автокредит` - зеленый (#10B981) цвет <br> 
💳 `Потребительский кредит` - фиолетовый (#8B5CF6) цвет <br> 
🏦 `Пенсионные накопления` - оранжевый (#F59E0B) цвет <br>

#### Все калькуляторы имеют одинаковую структуру:
- Тип "товара" (недвижимость/авто/цель кредита/тип пенсии)
- Тип кредита (стандарт/экспресс/другие варианты)
- Стоимость/сумма
- Первоначальный взнос (с бегунком)
- Форма отправки на Email
- Расчет с правой стороны
- Результаты в цветах соответствующих калькуляторов
